# 2. Reunião de Preparação (Planning) do Sprint 1

## 1. Revisão da Capacidade
**Velocidade Histórica:** Primeiro sprint - sem histórico anterior  
**Fatores de Limitação:** Equipe em fase de ambientação com as tecnologias  
**Capacidade Estimada para o Sprint:** 8 pontos  

## 2. Definição e Acordo da Meta
**"Implementar a base de dados para relatórios e estabelecer a estrutura inicial do sistema de avaliações"**

## 3. Seleção e Estimativa do Escopo

| Tipo | Título do Item | Estimativa |
|------|----------------|------------|
| Tarefa | SCRUM-60: Criar modelo de dados para relatórios no banco de dados | 5 pontos |
| Tarefa | SCRUM-56: Criar frontend para usuários avaliarem eventos e produtores | 3 pontos |

## 4. Decisões e Dependências

**Decisões Chave:**
- Utilizar PostgreSQL como banco de dados principal
- Implementar sistema de avaliações com notas de 1 a 5 estrelas
- Criar tabelas separadas para avaliações de eventos e produtores

**Dependências ou Riscos:**
- Necessidade de definir estrutura final dos relatórios com stakeholders
- Dependência do time de design para componentes de avaliação
